Editing the game files.

Cards.txt:
<cardId>:<Card Name>:<rarity>:<set folder name>:<texturename>
...

rarity:
	0 = common
	1 = uncommon
	2 = rare
	3 = mythic rare
	4 = Basic land
	5 = token
	6 = other
	
texturename: file must be a .jpg image, just put the name, without extensions.
	Recommended <set abbreviation><cardname>

Decks.txt:
<amount of decks>
<deckId>:<Deck Name>
...

<deckid>.txt:
<Deck Name> //For confirmation to check that the deck is accurate.
<cardId>:<amount of that card>
